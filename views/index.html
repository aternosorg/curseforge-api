Simple CurseForge API

This is a simple API for curseforge.com.
It mostly uses the same URL scheme as the CurseForge website. You can imagine this API like JSON glasses for CurseForge.
The API tries to directly fetch the data on request, but all requests are cached for 1 hour. A client-side cache is also recommended.



Currently supported methods

List projects

/{game}/{type}[?category={category}][&filter-game-version={game_version}][&filter-sort={field}][&page={page}]

/minecraft/mc-mods

Search projects

/{game}/{type}/search?search={search}

/minecraft/mc-mods/search?search=worldedit

Project information

/{game}/{type}/{slug}

/minecraft/mc-mods/worldedit

Project relations

/{game}/{type}/{slug}/relations/{relation}[?filter={field}][&page={page}]

/minecraft/mc-mods/journeymap/relations/dependencies

List project files

/{game}/{type}/{slug}/files[?sort={field}][&page={page}][&filter-game-version={version-id}][&filter-game-version-name={version-name}]

/minecraft/mc-mods/worldedit/files

File information

/{game}/{type}/{slug}/files/{file_id}

/minecraft/mc-mods/worldedit/files/2725648

Download information

/{game}/{type}/{slug}/download/{file_id}

/minecraft/mc-mods/worldedit/download/2725648

List categories

/other/{game}/{type}/categories

/other/minecraft/mc-mods/categories

List game versions

/other/{game}/{type}/versions

/other/minecraft/mc-mods/versions
